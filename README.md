# OCR-WITH-TESSERACT-WORDYPY-GAME⁉️
Final coursera project of PYTHON 3 SPECILIZATION.  
In this project i wrote a code to play the **WORDYPY** game and have the code give guesses on the basis  
of images of words provided by the given **GAME ENGINE**.  
The GAME ENGINE is provided by the coursera teachers.

# ADDITIONAL FILES IN THE PROJECT📁
- **word.py** = This module has the basics like DisplaySpecifications and the game engine.
- **words.txt** = Contains all the possible word combinations having 5 letters.
- **Roboto-Bold.ttf** = The photos and text of the output images.

# LIBRARIES UDED📚
- **wordy** = Given module to use the game engine and other.
- **PIL(PYTHON IMAGE LIBRARY)** = Its is a library used to perform modifications and many useful functions on the Images.

# SUB MODULES📗:- 
- **get_board_state** = Gives the current board state of the game to guess the next BEST GUESS.
- **make_guess** = Make a new guess on the basis of, rules of WORDYPY.
- **get_word_list** = Gives the current list of words that have been guessed.
- **get_display_spec** = Uses display specification if given by the user, else uses the default specifications.
- **DsiplaySpecification** = This is the default display specifications.
- **Image** = To use the image modification commands from the vast.
